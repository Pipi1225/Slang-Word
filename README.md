# README
## 1. Yêu cầu môi trường

Chương trình được biên dịch bằng Java 21.
Nên cần cài JDK hoặc JRE 21 trở lên để chạy.

## 2. Cách chạy chương trình

- Mở CMD hoặc PowerShell tại thư mục chứa file .jar, sau đó chạy:

`java -jar SlangDictionary.jar`

*(Nếu có nhiều phiên bản Java trên máy, hãy dùng đường dẫn đầy đủ.)*

## 3. Lưu ý

Tất cả dữ liệu được lưu trữ trong file đi kèm ở trong folder data.
